## 0. 硬性要求(v12 改立,用户钦定 2026-07-17 二轮,最高优先级,禁回退)

**棱齿轮 = 单一刚体折盘:折线是长在材料上的特征(与黑线槽重合),随自旋 300° 一起转
(GT = Jaap 官方解法页 `.tmp/gear/Gear Cube.pdf`,净 −60°/flip);只有自旋 ≡ 0°(mod 180°)
贴面,其余 60° 倍数静止时整件歪停(半边翘出面外、半边潜入槽喉)。**

- 真机裁决(用户实测):打乱态凹槽线离开棱线、折角是锐利 90° ⇒ 折线随材料转。
  照片中打乱的真机轮廓「炸毛」= 歪停的齿轮,这就是目标观感。
- 数学事实(不变):R(n̂,120°) 不把面平面映到面平面 ⇒ 刚体折盘只能在 0°/180° 贴面。
  v11 之前把「任意相位贴面」当铁律,由此发明的 fold-glide 履带模型(材料滑过固定折
  缝)在静止态与真机一致,但**转动中折缝不随材料转**,被真机证伪 ⇒ 废弃。
  fold 映射降级为**建构期一次性烘焙**(`buildGearPiece.bake`),运行时零逐帧顶点写。
- 齿冠(6 个 60° 饼楔 + 贴纸 + 黑线槽)整体挂 spin pivot,刚体旋转;黑线槽横跨
  dev q=0 = 烘焙折痕,「折线 ≡ 黑线」由构造保证。
- 连带净空(全部数值实证,`scripts/gear/rigid_check.mjs` + 测试锁):
  - 歪停的盘底会切入旧掌毂/背锥(−3.4)⇒ 两者下沉到 THROAT_OFF = PLATE_T·√2+1.8
    截距(下扫包络 ρ = a − PLATE_T·√2 之下 1.8 余量);
  - 刚体下潜 ~44 深 ⇒ 角块车床换「刚体扫掠回转体」轮廓(crownSweepInnerRadius,
    精确球界 ρ²+a² = p²+q²+d²),构造性挡住任意相位+过境;
  - 角贴纸 CORNER_POLY 按刚体扫掠重烘(φ0 ∈ {0,60,120} 全起始形状 × 双支 × 0.5°;
    折痕形状 180° 对称,60° 倍数静止相全覆盖):臂环 0 命中(净空数值见 §9 重烘记录)。
  - **悬挑双层(2026-07-19 用户钦定「别太保守」)**:过境材料在鳍/翼区最高只到
    h=125.2(SHELF gate 密扫),贴纸带从 128.5 起 ⇒ **可见贴纸+2.0 浅托板走完整 SVG
    原形 `CORNER_POLY_EXT`**(托板底 126.0 = 天花板+0.8),齿从悬挑刺下面滑过(真机
    同构);只有 9.8 深模切板保留过境裁剪形 CORNER_POLY(内缩 0.04 防共面)。孤立视图
    翼钮上方的缝 = 对接态齿冠贴折线料带的躺位,任何深度都不可填,拼装后由齿轮填满。
    oracle:mesh_check SHELF gate + rigid_check 双棱柱(过境 +0.32)+ vitest 双棱柱。
- 锁定:`tests/gear_geometry.test.ts` — 相位 0 逐顶点共面 + **相位 120° 齿尖离两面

## 3. 中心臂设计(derive3 第一版曾按「边块轴座」做,被数值扫否决)

**归属排他论证**:
- ❌ 挂齿轮件(orbit pivot):面层轮翻面 90° 时其面板扫过赤道轮鼓出面板的盘面区
  (t=0.5 实测 232 点穿透);面板径向带 [0.30H,0.42H] 与赤道轮盘面穿面圆盘(半径
  ~0.385H、圆心正对侧面中心)必然相交,物理不可避。
- ❌ 挂角块:括号位于棱中线,不在角块任何 slab。
- ✅ 挂中心块:臂随中层走位。三条相对运动全部可清:
  ① **臂 ⊂ 赤道环咬合管**(max hypot(|壳半径−RING_R|, y) < R_BITE)→ 对全部角块
     构造零穿模(保护齿轮的同一根管);
  ② 面层轮绕轴壳层(xz 半径 × 高度)与臂壳层不相交 ⟸ 臂径向外沿 ≤ ~0.375H
     (轮触及壳半径 r 处的最低高度 y_min(r) = 184.4 − r + 2wS2,臂须在曲线下方);
  ③ 臂与赤道轮同属中层,刚性;绕自面转动时臂随面走,与本面齿轮刚性。

**尺寸**(面内坐标,r̂ = 面心指向该棱,单位 u,H=128):
- 臂整体:径向 r ∈ [38.4, 48](0.30–0.375H),切向 ±24(0.1875H),体厚 5,贴纸浮高 ~3.1
- 横杆:r ∈ [44, 48] 全切向;方脚 ×2:r ∈ [38.4, 44],|s| ∈ [12, 24]
- 切向 ±24 上限来自角块对角小舌壳层(hypot(24, 131.5)=133.7 < 舌尖壳 136,余 2.3)
- 比图纸(0.30–0.47H)径向压短:外沿被面层轮 splat 扫掠曲线钉死(0.375H),同根因

## 5. 物理极限备忘(为何不能百分百复刻)

轮对轮中途在 ~53u 联合半径盘面相交;真魔方靠齿槽啮合 + 塑料弹性,而模拟器相位是自由状态
(每环独立 mod 3,全组合可达),不能赌啮合 → 齿尖半径 < 相交半径(上限 0.362H)。
扇宽、柄连续性、舌深三处差距全部源于此,属同一根因。

## 6. 折棱伞冠齿轮(2026-07-16 用户纠正,推翻 45° 平盘轮)

**目标**(用户原话):还原态下齿轮两半所在平面**垂直**——绿半与绿角块共面(F 面平面内)、
白半与白角块共面(U 面平面内),毂为棱上圆形。

**形状推导**:
- 「对折硬币」(两半各自完全平)被排除:自旋 60° 后折线不再对齐棱(刚性旋转无解,
  且状态模型 spin=60°/flip 已 cstimer 锁死)。形状必须 60° 自旋不变。
- 唯一解 = **六片伞**:6 颗齿各为平板,板面切于「锥尖=棱中点 E、半角 45°、轴=径向 n̂」
  的圆锥;正对两面的 ±90° 齿板面**精确落在面平面内**(锥尖在棱上 ⇒ 切平面 = 面平面);
  相邻齿板在齿槽谷线处折 ~41°(法线夹角 acos(0.75));毂 = E 处圆顶。
- 伞冠 ⊂ 锥 ⊂ {两面平面内侧} ⇒ 任意自旋角都不冒出面(还原态齐平)。
- 贴纸 = 每齿板面一张(inset 平贴),3 绿 3 白;随齿自旋 → 混色扇 ✓ cstimer facelet 一致。

**间隙全解**(轮心从环圆 130.3 挪到棱点 181,一切变宽松):
- 伞冠 ⊂ ball(≈56) @ E;面层冠轨道圆 {r=128, y=128} vs 赤道冠 {r=181, y=0} 最近 138.6
  ≫ 2·56+2 → **球包围直接够**,自由相位天花板消失,扇面可做满 0.43H。
- 角块咬合改为 **3 只棱环面**(TorusGeometry(H√2, R_BITE_E≈59))——赤道冠扫掠 ⊂ 绕棱圆环管。
- 臂保护:角块内对角材料仍会扫过臂壳层 → 每轴加 **washer 环刀**(绕轴矩形回转体,
  截面半径 [120,136] × |轴| ≤ ~57)专切臂扫掠带;washer 不触及舌尖(hypot(45,128.5)>136)。
- 角舌尖 ≈ 0.35H(vs 图 0.36–0.38H ✓);扇道宽 ±0.48H 容 0.43H 扇 ✓;臂可回到图纸 0.46H。

**实现**:锥尖 E = slot 方向 · H√2;facet 基 (ĝ=(û−n̂)/√2, ŵ=m̂×ĝ, m̂=(û+n̂)/√2),
û=cosφ ê+sinφ t̂;每齿 = facet 平面内梯形齿多边形 ExtrudeGeometry(depth 内向)+ 贴纸
extrudeOntoFace;底部黑锥壳兜底;毂球。engine 其余零改(自旋轴过原点含 E)。

## 7. 定稿细化(2026-07-16 二轮,用户钦定)

- **齿形 = 等腰梯形**(TOOTH_HALF_ANG=11°,齿 22° / 隙 38°):直腰的延长线过圆心,
  上下底为弦(TOOTH_ROOT=16 → FAN_R=0.43H);任意半径处隙弦 2r·sin19° > 齿弦
  2r·sin11° ⇒ 间隔大于两条底边。贴纸 = roundCorners(2.5)+offsetInward(2)。
- **角块雕刻 = 冠扫掠 lathe**(替代过肥的球管):自旋扫掠壳在 (ρ,h) 坐标是 45° 锥壳
  (ρ=(a+d)/√2, h=(a−d)/√2),绕转轴回转 → inner(Y)=EDGE_R−(Y≤ρ₀?H_max:FAN_R√2−Y)−1,
  H_max=(FAN_R+PLATE_T)/√2≈43.2、半高 SWEEP_RHO=(FAN_R+3.1)/√2≈41.1。
  **LatheGeometry 闭合轮廓必须 (r,y) 逆时针**(底→外壁→顶→内壁),顺时针 = brush
  内外翻转 → CSG 喷碎片毛刺(第一轮毛刺根因)。
- **自旋 300°/180°**(GT:Jaap 官方解法页,推翻 issue #32 时期的 480°):真机齿比;
  净 −60°/flip。Z3 相位不破:齿轮回到原槽需 4k flip,4k·300 ≡ 120k(Jaap 亲证
  "two complete rounds ... twists them by 120°"),同槽位可见朝向只有 3 档;±60° 奇偶
  被槽位奇偶(净 flip mod 2)奴役,spin pivot 累加真实旋转天然与离散态一致。
  R 的阶 = 12(面 2 × 中层 4 × 相位 3 的 lcm)不变。

## 8.3 弯硬币轴帽(2026-07-16 四轮,用户实拍钦定「不是圆锥,是硬币掰弯 90°」)
